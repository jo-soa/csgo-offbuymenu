A SourceMod plugin for CS:GO that allows players to purchase weapons not available in the default buy menu using chat commands.

`!usp` – Receives a USP-S (CT only, must be alive and once per round)  
`!m4` – Buys an M4A1-S (CT only, must be alive and in the buy zone, $2900)  
`!mp5` – Buys an MP5-SD (must be alive and in the buy zone, $1500)  
`!r8` – Buys an R8 Revolver (must be alive and in the buy zone, $600)  
`!cz` – Buys a CZ75-Auto (must be alive and in the buy zone, $500)
